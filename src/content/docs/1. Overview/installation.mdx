## Font Family and Typography

The components use the [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) font family, which is imported from Google Fonts. This font family is used throughout the components to ensure a consistent and visually appealing typographic style.

```tsx filename=layout.tsx
import { Plus_Jakarta_Sans } from "next/font/google";
// ...

const font = Plus_Jakarta_Sans({ subsets: ["latin"] });

export default function RootLayout() {
    // ...
    return (
        // ...
        <body className={font.className}>{/* ... */}</body>
        // ...
    );
}
```

                ## Typescript Configuration

                The components are written in TypeScript, which provides type safety and improved code quality. The TypeScript configuration is set up to support the components and ensure that the codebase is consistent and maintainable.

                ```json filename=tsconfig.json
                {
                    "compilerOptions": {
                        // ...
                        "paths": {
                            // ...
                            "@/colidy-ui/*": ["./src/components/colidy-ui/*"],
                            "@/utils/*": ["./src/utils/*"]
                        }
                    }
                    // ...
                }
                ```
